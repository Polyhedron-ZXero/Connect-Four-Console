Connect Four
-------------
This program uses symbol characters in Code Page 936.
Please make sure that the operating system supports
Code Page 936 (GB2312 - ANSI/OEM Simplified Chinese)
in order to display the Unicode characters correctly.

If your computer does not support Code Page 936 and
displays many "?"s when running the program, there
are 5 screenshots provided in order to show you what
the game should look like.
